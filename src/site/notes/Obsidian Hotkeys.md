## Basic

- Settings: `alt-,`
- Enter a command: `alt-P`
- Find or create a note: `alt-O` (Letter O)
- Search: `cmd-shift-F`

## Templates/Templater
- Create a note from a template: `^T`
- Evaluate templater code in current note: `alt-R`

## Other
- Toggle sliding pane (Andy Matuschak) mode: `ctrl-alt-T`
